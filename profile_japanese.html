  • Agro Detect: 深層学習による稲病気分類

    Python - Keras, Streamlit; Android - Java

    • 深層学習を使用した稲病気分類。Paddy Doctorデータセットを使用して、Octave ConvolutionやSqueeze-Excitation Networksなどの新しい技術を含むカスタムCNNアーキテクチャを利用。最終的な精度97.59を達成。

    • 作物の病気を特定するために画像をアップロードできるAndroidアプリケーション用の展開可能な機械学習パイプラインを作成。
